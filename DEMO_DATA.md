## 使用说明

### 视频演示流程

1. **打开产品网站**
2. **复制原始简历** → 粘贴到 "Paste Your Resume"
3. **复制 Job Description** → 粘贴到 "Job Description"
4. **点击 "Optimize My Resume"**
5. **展示 ATS 评分**（从 35 涨到 82）
6. **展示关键词匹配**（绿色 vs 红色）
7. **展示优化后简历**（干净格式）
8. **切换到 Cover Letter 模式**
9. **生成求职信**
10. **点击 Download PDF**

### 展示要点

| 画面 | 强调 |
|------|------|
| ATS 评分 | "从 35 涨到 82" |
| 关键词匹配 | "找到 12 个缺失关键词" |
| 优化后简历 | "格式清晰，ATS 友好" |
| 求职信 | "30 秒生成专业求职信" |
| PDF 导出 | "一键下载" |

### 时间控制

| 步骤 | 时间 |
|------|------|
| 粘贴简历 | 3 秒 |
| 粘贴 JD | 3 秒 |
| 点击优化 | 1 秒 |
| 展示结果 | 15 秒 |
| 求职信 | 10 秒 |
| PDF 导出 | 3 秒 |
| **总计** | **35 秒** |
